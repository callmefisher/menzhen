# 按方开药 - 方剂信息展示优化

## 日期
2026-02-26

## 目标
在处方弹窗的"按方开药"模式中，选中方剂后自动展示功效、主治、备注信息，美化页面布局。

## 变更范围
仅前端：`web/src/components/PrescriptionModal.tsx`

后端已返回完整数据（effects, indications, notes），无需改动。

## 布局设计

选中方剂后，在药物列表上方展示信息卡片：

```
┌─────────────────────────────────┐
│  方剂搜索: [_________ ▼]        │
│                                 │
│  ┌─ 方剂信息 ─────────────────┐ │
│  │ 功效: 补气养血...           │ │
│  │ 主治: 气血两虚...           │ │
│  │ 备注: 服用注意事项...       │ │
│  └─────────────────────────────┘ │
│                                 │
│  ┌─ 药物列表 ─────────────────┐ │
│  │ 药名  |  用量  |  备注     │ │
│  │ 黄芪  |  30g   |          │ │
│  │ 当归  |  15g   |          │ │
│  └─────────────────────────────┘ │
│  总剂数: [7]                    │
│  医嘱: [____________]           │
└─────────────────────────────────┘
```

### 卡片样式
- Ant Design `Descriptions` 组件，浅色背景 + 圆角边框
- 未选方剂时不显示
- 字段为空则隐藏该行

### 搜索下拉优化
展示更多信息：方剂名 + 功效 + 主治前缀

## 数据流
无新增 API。在前端 state 中保存选中方剂的完整信息即可。
