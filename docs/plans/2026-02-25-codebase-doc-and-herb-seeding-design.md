## 二、药物/方剂预录入脚本（scripts/seed-herbs-formulas.sh）

### 运行方式

Shell 脚本，通过 HTTP 调用后端 API，复用现有 DB→AI回退→自动保存机制。

### 流程

1. 登录获取 JWT token（`POST /api/v1/login`）
2. 逐个调用 `GET /api/v1/herbs?name=药物名`，触发后端自动录入
3. 逐个调用 `GET /api/v1/formulas?name=方剂名`，同理
4. 每次请求间隔 2-3 秒，避免 AI API 限流
5. 记录进度和日志

### 药物/方剂覆盖范围（各 ≤800 种）

| 来源 | 覆盖内容 |
|------|----------|
| 伤寒论 | 经方用药 + 方剂 |
| 温病条辨 | 温病方剂 + 用药 |
| 金匮要略 | 杂病方剂 |
| 神农本草经 | 基础药物 |
| 历代名家 | 张仲景、孙思邈、李时珍、叶天士、吴鞠通、朱丹溪、刘完素、张景岳 |
| 临床常用 | 补中益气汤、六味地黄丸、逍遥散等 |

### 脚本功能

- **断点续传**：记录已处理项到进度文件，重启后跳过已成功项
- **可配置**：API 地址、用户名密码通过环境变量或参数
- **日志**：成功数、失败数、跳过数统计
- **参数**：`--herbs-only` / `--formulas-only` / `--dry-run`
- **进度文件**：`/tmp/seed-herbs-formulas-progress.log`
